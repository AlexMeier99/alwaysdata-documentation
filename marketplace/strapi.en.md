[Strapi](https://strapi.io/) is a fully customizable [CMS](https://en.wikipedia.org/wiki/Content_management_system) built with [Javascript](https://en.wikipedia.org/wiki/JavaScript).

## Installation

The last version of *Strapi* is auto-installable via the administration interface: **Web > Sites > Install an application**.

- [Install script](https://admin.alwaysdata.com/site/application/script/181/detail/)
